# Logger Service

Comprehensive logging system with feature-flag-based filtering and local storage persistence. Enables debug logging for various application layers including Redux, services, API calls, and cache operations.

## Responsibilities

- Log messages with different severity levels (info, warn, error, cache)
- Feature-flag controlled debug output for different modules
- Local storage persistence of logs for debugging
- Log subscriber notifications for real-time monitoring
- Export and clear log history
- Format messages with emoji prefixes for visual distinction
